# Pensa

Pensa és un llenguatge de programació educatiu en català.

## Executar

```bash
python pensa.py examples/hola.pensa
```

## Mode explicatiu

```bash
python pensa.py examples/condicions.pensa --explica
```

## Exemple

```text
guarda 20 dins edat

si edat >= 18

    mostra "Adult"

si no

    mostra "Menor"

acaba
```

## Instruccions disponibles

- mostra
- guarda
- si
- si no
- acaba
- repeteix
